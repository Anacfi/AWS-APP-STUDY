# AWS Study

PWA para estudiar certificaciones de AWS (empezando por Cloud Practitioner). Instalable en iOS desde Safari ("Agregar a pantalla de inicio").

## Desarrollo

```
npm install
npm run dev
```

## Build y deploy a GitHub Pages

```
npm run deploy
```

Esto compila y publica `dist/` en la rama `gh-pages`. La app queda en `https://anacfi.github.io/AWS-APP-STUDY/`.
