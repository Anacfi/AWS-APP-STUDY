# Cómo entregar tus notas

Deja aquí tus PDFs o notas (.pdf, .md, .txt) nombrados así:

```
<Lección> - <Concepto>.pdf
```

Ejemplos (Cloud Practitioner, categoría Fundamentos):

```
Modelos de precios de AWS - On-Demand.pdf
Modelos de precios de AWS - Reserved Instances.pdf
AWS Well-Architected Framework - Seguridad.md
```

El nombre de la lección y el concepto deben coincidir exactamente con los que ya existen en la tabla `lessons`/`concepts` (los que están en `supabase/migration_002_certs_and_concepts.sql`). Si el nombre no coincide, dilo y ajustamos la base de datos o el nombre del archivo.

Cuando subas archivos aquí, avísame y los leo, extraigo el contenido y los subo a la columna `concepts.content` en Supabase.

Esta carpeta no se sube a GitHub (son tus notas de estudio personales).
